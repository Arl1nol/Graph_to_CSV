# Graph to CSV Digitizer

Extract data points from graph images and save them as CSV files.

## How it works
1. Click on the line color you want to extract
2. Set color sensitivity threshold
3. Click bottom-left then top-right corners of the graph
4. Enter real-world axis values
5. Choose how many output points you want
6. Get a clean `graph.csv` file
